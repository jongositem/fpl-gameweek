About This FPL Application

Overview

This is a Fantasy Premier League themed web application built with Flask, using the official Premier League styling. The application demonstrates modern web development practices including:

  • Official Premier League CSS theme integration
  • FPL color scheme (#37003c and #00ff87)
  • RESTful API endpoints for fantasy football data
  • JavaScript for dynamic interactions
  • Template inheritance with Jinja2
  • Error handling and custom error pages

Technical Stack

Backend

  • Python 3.13+
  • Flask 3.1.2
  • Port 8090

Frontend

  • Official Premier League CSS
  • HTML5 & CSS3
  • Vanilla JavaScript
  • FPL Theme Colors

Features

  • Responsive Design
  • API Integration
  • Form Handling
  • FPL Themed UI

API Documentation

GET /api/data

Returns information about available API endpoints for FPL data

POST /api/data

Accepts JSON data from FPL managers and returns a confirmation response

Content-Type: application/json

Example payload:

{
  "name": "FPL Manager",
  "message": "My team is performing well this season!"
}
